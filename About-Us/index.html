What we set out to accomplish....

Create sleep tracker that:

    Allowed the user to create an alarm
    Determine sleep quality through measureable metrics
    Improve the quality of sleep for users

Our key features...

    On-boarding process for the user
    A homepage that shows a graph of users nightly sleep over time
    Ability to manually input sleep data by date. (through a digital clock styled interface.
    Ability to edit a sleep entry
    Ability to rate sleep quality through a range of emojis
